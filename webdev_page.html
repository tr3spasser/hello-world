Website Developement

Introduction

These days most people use website builders to create their websites as have I. However, during the module, we were introduced to html, css and javascript. I figured it would be interesting to try writing the website programmes myself. Therefore, I used Visual Studio Code to programme the website.


Programming

I took me a while to understand the basic syntax of html and css. Following cheatsheets and examples online, I was able to create lists and write text in different sizes. With CSS, I learnt how to create boxes for text, change the colours of objects and insert pictures.


Github

Creating a github account was simple. However, getting my head around how Github works is difficult. Also I was confused about how to link different files to each other. For example, if I have files in folders what path should I put to call that file if my other file is in a different folder. I also kept making the mistake of updating many files in quick sucession and have some of the uploads fail. However, I think Github is really useful for collaboration and storing project files on. I do want to continue using it in the future.
