2D Graphics Editing

Introduction

In this module, we were introduced to two 2D graphics editing softwares. Inkscape and GIMP. Both are free to download, open-source softwares. However, Inkscape is a vector graphics editing software while GIMP is a raster graphics editing software.


In vector graphics, images are created from geometrically defined points, lines and shapes on a cartesian plane.

In raster graphics, images are represented by a grid of square pixels. Each pixel contains unique colour and tonal information that come together to form an image.


Inkscape

Inkscape is a useful tool to prepare vector graphics files for import into a CAD software. It is also useful for preparing our designs from the CAD software for laser cutting.
